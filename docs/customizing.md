# Customizing

## 文言を変える

多言語の文言は `i18n/locales/` にあります。  
まずは対象の言語ファイルを開いて、必要なキーを書き換えてください。

## 言語を追加する

対応言語を増やしたいときは `nuxt.config.ts` の `i18n` 設定を更新します。

確認ポイント:

- locale の一覧
- デフォルト言語
- メッセージファイル

## SEO の URL を変える

SEO 用のベース URL は `nuxt.config.ts` の `i18n.baseUrl` で管理します。

例:

- 本番ドメインが変わった
- 検証環境から本番環境に切り替えた

## ダミー API を差し替える

開発中のデータは `server/api/` にあります。  
本物の API に切り替える前に、まずはここで画面の動きを確認できます。

## 本番前に見直すもの

| 項目 | 確認内容 |
|---|---|
| ブランド名 | 仮の名前が残っていないか |
| 文言 | サンプル文言が残っていないか |
| SEO | `baseUrl` が正しいか |
| API | ダミーデータのままになっていないか |
