# Getting Started

## 対象読者

このドキュメントは、Nuxt にまだ慣れていない開発者向けです。

## 事前に必要なもの

| 項目 | 内容 |
|---|---|
| Node.js | 一般的には LTS 版を使う |
| pnpm | このリポジトリで使うパッケージマネージャー |

## 開発を始める手順

1. パッケージをインストールする
2. 開発サーバーを起動する
3. ブラウザで確認する

```bash
pnpm install
pnpm dev
```

## 開発の流れ

| やること | 主に見る場所 |
|---|---|
| 新しいページを作る | `app/pages/` |
| UI 部品を追加する | `app/components/` |
| 表示文言を変える | `i18n/locales/` |
| API の仮データを変える | `server/api/` |

## エラー確認

型エラーを見たいとき:

```bash
pnpm typecheck
```

本番ビルドを試したいとき:

```bash
pnpm build
pnpm preview
```

## つまずきやすいポイント

| 困ること | 確認すること |
|---|---|
| `pnpm` がない | `pnpm` をインストールしているか |
| 起動しない | `pnpm install` を実行したか |
| 文言が変わらない | `i18n/locales/` の対象ファイルを編集したか |
| API の値が変わらない | `server/api/` の対象ファイルを編集したか |
